# Percentilová kalkulačka

Jednoduchý Python skript pro výpočet percentilového rozmezí
výšky a hmotnosti dítěte podle věku a pohlaví.

## Spuštění
1. Ujisti se, že máš Python 3.x
2. Spusť:
   python calculator.py

## Vstupy
- věk (měsíce)
- výška (cm)
- hmotnost (kg)
- pohlaví (M/F)

## Data
- percentilová data jsou uložena v souboru `percentiles.csv`

## Výstup

Program vypíše, zda je vstupní hodnota pod,
v rámci nebo nad normou pro daný věk.
